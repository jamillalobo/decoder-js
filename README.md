
# Decodificador de JavaScript

This project was made for the final project of an Alura course. It was used concepts of JavaScript, CSS and HMTL.

## Encript Logic

the logic behind the encript is to change the vogals (to decript is the opposite):

`A letra "e" é convertida para "enter"`

`A letra "i" é convertida para "imes"`

`A letra "a" é convertida para "ai"`

`A letra "o" é convertida para "ober"`

`A letra "u" é convertida para "ufat"`
